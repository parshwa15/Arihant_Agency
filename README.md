# Dealer Dashboard v3.1

Fixes & features:
- Data reliably renders after Search
- Month defaults to "ALL"; month filter optional
- Top badges show Total Items, Dealer Code, Month
- PDF export drops Dealer Code/Dealer Name columns; adds them to the header instead
- Dynamic columns supported across dashboard, Excel and PDF

Run:
```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
